# Markdown Studio

A professional local-first Markdown editor, viewer, and formatter built with React, Vite, CodeMirror, and Prettier.

## Features

- Split editor and live preview workspace
- GitHub-flavored Markdown support
- CodeMirror editor with syntax highlighting
- One-click Markdown formatting with Prettier
- Toolbar for headings, lists, tasks, quotes, links, images, tables, and code
- Local document manager with auto-save
- Search, duplicate, delete, and import Markdown files
- Templates for README, blog posts, meeting notes, API docs, and changelogs
- Export to `.md`, `.html`, copy HTML/Markdown, and print/save as PDF
- Dark/light mode and responsive layout
- Word count, character count, line count, and reading time

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Product Direction

The MVP is frontend-only and local-first. A paid version could add cloud sync, team workspaces, custom PDF themes, branded exports, publishing, and AI-assisted rewriting or format conversion.
